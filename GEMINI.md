# Brand Brief: Dr. Nor Faizal
- **Persona:** Oxford-trained Neurosurgeon. Precision-driven. Human-first.
- **Goal:** Convert clinical expertise into patient consultations.
- **Language Policy:** Use "Bahasa Melayu" for social reach; "English" for medical authority.
- **Code Style:** Hugo Static Site (minimalist). Zero bloat.
- **Instruction:** When asked to "Optimise for reach," translate core concepts into simple Bahasa terms.

## Translator & Editor SOP
- **Logic:** When processing a file, always generate two versions:
    1. **The Edit:** Improve the source file for medical precision, discipline, and authority.
    2. **The Translation:** Create a mirror file in the opposite language (EN <-> BM).
- **Naming Convention:** - If source is `*.bm.md`, translation is `*.en.md`.
    - If source is `*.en.md`, translation is `*.bm.md`.
- **Quality Control:** Medical terminology must remain precise in both languages. Use "Saraf" for nerves, "Tulang Belakang" for spine, but keep complex Latin terms (e.g., *Glioblastoma*) consistent.
- **Tone & Refinement:** For Bahasa Melayu content targeting social reach, refine the language to be simpler, more direct, and conversational ('Bahasa moden'). Avoid overly formal or academic phrasing. Use English medical terms where they are more common or precise (e.g., *Glioblastoma*, Simpson Grade, Neuro-navigation) to maintain authority, but explain them in simple BM.
